# Learning Code

Flutter 학습 실습 코드를 모으는 공간입니다.

## 권장 구조

```text
lib/learning/
  labs/
    lab_t001_counter_basics.dart
    lab_t002_layout_constraints.dart
    lab_t003_todo_state.dart
  shared/
    # 여러 실습에서 반복해서 쓰는 작은 위젯이나 모델
```

## 작성 규칙

- 한 파일은 하나의 실습 주제에 집중합니다.
- 실습 화면 위젯 이름은 파일명과 맞춥니다.
- 여러 실습에서 재사용할 코드가 생기기 전까지는 과하게 분리하지 않습니다.
- 완료된 실습은 삭제하지 말고 다음 실습과 비교할 수 있게 남겨둡니다.
